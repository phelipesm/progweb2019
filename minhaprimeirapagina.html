A Linguagem HTML

HTML(abreviação para expressão inglesa HyperText Markup Language,que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web.
Documentos HTML podem ser interpretados por navegadores.

Até este momento,vimos os seguintes recursos da linguagem:

Recurso Tags associados Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar lista de itens ordenados
<ol> e <li> Criar lista de itens não ordenados
Imagens <img> Inserir imagens
